使用github page + Hexo 紀錄code筆記

近期開始學django
想要有個可以記錄的空間但又不願使用常見的部落格
因緣際會下得知到github page

免費有好有壞

好處：

  1. 屬開源網站，不易壞
  2. 自由度高
  3. 免費

缺點：

  1. 流量限制吧
  2. 圖片盡量以上船圖床使用外部連結較佳

關於github page專案設定參考[教學] 使用 GitHub Pages + Hexo 來架設個人部落格

安裝完前置動作可回來看，或繼續依照上述連結執行下去也是OK

以下重點一些指令及筆記

安裝必要套件

  • Node.js
  • git
    安裝Hexo
    建立網站包 & 安裝git部署
    git佈署套件參考：Hexo官網

由於是使用md語法撰寫文章推薦使用Notepad++ 撰寫
倘若有自己使用的也是可行

部落格相關設定在cmd資料夾下的 _config.yml

_config.yml 檔案中6~12行可設定部落格基本資料

  • title
  • subtitle
  • description
  • keyword
  • author
  • language: zh-TW
  • timezone

hexo實行指令

啟動server 預設 http://localhost:4000
