default_theme_board

不知道大家是不是都不愛 landscape
在網路上看了滿多解決的文章
大多是用套用新主題模板本身富含的功能
也有很多檔案需要修改的
第二種就比較因人而異
每個人放檔案的地方都有所不同

卡了快一周
才正式回來面對這在Hexo default theme 留言板問題
(前些天肚痛炸)

比較常看到就是disqus
這個網站提供整合留言功能
disqus註冊流程根據這篇教學跟著做

幾個步驟即可使用

到前面設定好的網站進入管理(Manage) -> Installation
在 disqus， Hexo 並沒有被列在上面
在最下面
會跳轉頁面，並有一段程式碼

官方提到這段程式碼就是安插在想要顯示的地方

不過經過多次嘗試，就執行下面流程即可

1.在主題的 _config.yml新增disqus_shortname

檔案路徑：
./node_modules/hexo-theme-landscape/_config.yml
預設主題默認：enabled=ture
因此只需在最底下加入

2.在after-footer.ejs加入disqus的條件判斷

搜尋 after-footer.ejs 在檔案中加入config.disqus_enabled的條件判斷

3.將disqus代碼加入到每篇貼文下方

檔案路徑：
./node_modules/hexo-theme-landscape/layout/post.ejs
直接將最<div id="disqus_thread"></div>開頭的這些代碼貼到post.ejs內容下方(如圖)

儲存後就完成留言板的啟用了

另外可能會想要更改語言設定

參考本篇教學
拿掉 disqus_thread 中這段代碼 6~11行的多行註解
並在這當中的內容作修改
